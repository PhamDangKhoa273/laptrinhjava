# BICAP Documentation

This directory is organized by documentation type so the repository remains easy to review.

## Structure

| Folder | Purpose |
|---|---|
| `agents/` | Agent context and working skills |
| `architecture/` | System architecture, project context, module map |
| `business/` | Business rules and requirement traceability |
| `api/` | API contracts, Postman guides, API testing notes |
| `modules/` | One markdown spec per business/technical module |
| `operations/` | Runbooks, security checklists, deployment/demo operations |
| `testing/` | Verification logs, test cases, benchmark evidence |
| `evidence/` | Evidence artifacts retained for submission/review |
| `archive/` | Historical phase/audit/handoff notes |
