# Archive

Tài liệu lịch sử đã được thay thế bởi tài liệu mới trong cấu trúc đánh số 00..09.

Chính sách lưu trữ:

- Giữ tối thiểu 1 release cycle (~3 tháng) trước khi xoá vĩnh viễn.
- Tài liệu trong `_archive/` không phải là authoritative — chỉ tham khảo lịch sử.
- Phase reports, audit reports cũ, handoff documents thuộc về đây.
- Các redirect stubs cho legacy AGENTS.md anchors KHÔNG sống trong `_archive/`; chúng nằm tại path gốc với `status: superseded`.
