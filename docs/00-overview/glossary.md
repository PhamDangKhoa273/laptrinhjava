# BICAP Glossary

Bảng tra cứu nhanh các thuật ngữ. Phiên bản đầy đủ với rejected synonyms ở [`02-domain/ubiquitous-language.md`](../02-domain/ubiquitous-language.md).

| Term (canonical EN) | Vietnamese | Mô tả ngắn |
|---|---|---|
| Farm | Trang trại | Đơn vị sản xuất nông nghiệp |
| Farm Manager | Quản lý Trang trại | Vai trò vận hành Farm |
| Retailer | Nhà bán lẻ | Vai trò mua sản phẩm từ Farm |
| Driver | Tài xế vận chuyển | Vai trò thực hiện giao nhận |
| Shipping Manager | Quản lý Vận chuyển | Vai trò điều phối chuyến hàng |
| Guest | Khách | Người dùng công khai chưa đăng nhập / đã đăng ký guest |
| Season | Mùa vụ | Một chu kỳ canh tác trên một trang trại |
| Batch | Lô hàng | Đơn vị sản phẩm gắn nguồn gốc Season |
| Listing | Đăng sàn | Sản phẩm được đưa lên sàn giao dịch |
| Order | Đơn hàng | Yêu cầu mua hàng từ Retailer |
| Deposit | Tiền đặt cọc | Khoản trả trước khi đặt mua |
| Shipment | Chuyến hàng | Đơn vị vận chuyển một order |
| Subscription | Gói dịch vụ | Gói trả phí cho phép Farm dùng hệ thống |
| QR Code | Mã QR | Mã truy xuất nguồn gốc sản phẩm |
| IoT Alert | Thông báo IoT | Cảnh báo từ cảm biến (nhiệt độ/độ ẩm/pH) |
| Trace | Truy xuất nguồn gốc | Quá trình theo dõi sản phẩm qua các giai đoạn |
| Proof | Bằng chứng blockchain | Bản ghi trên VeChainThor xác nhận sự kiện trace |

Thuật ngữ kỹ thuật (RBAC, JWT, EARS, SSOT...) ở [`02-domain/ubiquitous-language.md`](../02-domain/ubiquitous-language.md).
